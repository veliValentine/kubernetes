# Dummy site kubernetes controller

## Start
1. `npm i`
2. `npm start`

## Start dev
1. `npm i`
2. `npm run dev`

## Env variables

| key            | default value | info                                                              |
| -------------- | ------------- | ----------------------------------------------------------------- |
| `PORT`         | `3000`        | `Application port`                                                |
| `NODE_ENV`     | `prod`        | `prod` OR `dev`                                                   |
| `BEARER_TOKEN` | undefined     | Bearer token for dummy site kubernetes item api. `Bearer <TOKEN> ` |
